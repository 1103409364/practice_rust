# 文件浏览器应用

一个使用 Rust 和 egui 框架开发的简单文件浏览器应用。

## 功能特点

- 文件系统导航
- 文件内容预览
- 中文界面支持
- 深色/浅色主题适配

## 系统要求

- Rust 1.70.0 或更高版本
- 支持的操作系统：Windows、macOS、Linux
